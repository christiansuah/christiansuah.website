2) What is the difference between HTML and XHTML?

HTML (Hypertext Markup Language) and XHTML (Extensible Hypertext Markup Language) are both markup languages used to create web pages, but they have some key differences: Syntax Rules: HTML: More lenient with syntax rules. Browsers can often correct minor errors in HTML code. XHTML: Stricter syntax rules. It follows XML (Extensible Markup Language) standards, meaning all tags must be properly nested, closed, and written in lowercase. Error Handling: HTML: Browsers are forgiving and will try to render the page even if there are errors. XHTML: Browsers will not render the page if there are syntax errors. Document Structure: HTML: Doctype declaration is optional. XHTML: Requires a doctype declaration and the xmlns attribute in the tag to specify the XML namespace. Element and Attribute Naming: HTML: Tags and attributes can be written in uppercase or lowercase. XHTML: Tags and attributes must be written in lowercase. Empty Elements: HTML: Empty elements (like
